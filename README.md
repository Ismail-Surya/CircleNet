# CircleNet

CircleNet is a full-stack social networking platform inspired by classic social media applications.

## Tech Stack

### Frontend

- React
- TypeScript
- Vite
- Bootstrap 5
- React Router
- Axios

### Backend

- Java 8
- Spring Boot 2.7.x
- Spring Data JPA
- Spring Security
- MySQL
